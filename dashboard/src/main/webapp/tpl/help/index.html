General

NATL (Nitrogen Automation Test Lab) is a website designed for test automation execution for WebDriver(Selenium 2) test cases written in JAVA.


The system enables you to:

  • Execute test cases on slaves with different browsers any time, anywhere
  • Troubleshoot test failures with detailed test reports
  • Oversee the automation coverage with historical execution results

Integration Requirements

To make full use of the system, we need to make sure following items are ready:

  • The automation code is written in JAVA
  • The automation code is annotated by TestNG
  • The source is integrated with MAVEN, and it's preferred that the project source structure follows MAVEN recommendations.
